# Lumora

**Independent Design & Engineering Studio**

A landing page that feels like a quiet statement. Smooth scroll pulls you through a dark intro loader into a full-bleed hero where a liquid brush cursor reveals a second world underneath. Giant watermark typography anchors every section while spring-physics animations bring each element to life.

---

### What's inside

- **Liquid cursor reveal** — brush your pointer across the hero to uncover a hidden layer
- **Intro loader** — count from 000 to 100, then the curtain lifts
- **Line-by-line text reveals** — headlines emerge word by word as you scroll
- **Auto-cycling carousel** — portfolio cards rotate on their own
- **Live clock** — real-time date and time in the header
- **Count-up stats** — numbers animate when they enter the viewport
- **Full-screen nav overlay** — dark menu with staggered link animations
- **Request modal** — clean contact form with success state
- **Adaptive grid** — scales from mobile to ultrawide without breakpoints

### Stack

Single HTML file. No build tools. Lenis smooth scroll, CSS custom properties, vanilla JS, `rem`-based fluid typography.
